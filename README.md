# ToDoList

A CLI-based task management application written in Python.

This project implements full CRUD functionality using Object-Oriented Programming (OOP) principles and CSV file persistence. Users can add, view, search, edit, and delete tasks, with all changes saved to disk.

Built to practice data modeling, file I/O, error handling, and menu-driven program design.

---

## Features

- Add tasks  
- View tasks  
- Search tasks  
- Edit tasks  
- Delete tasks  
- CSV file persistence  

---

## Concepts Used

- Object-Oriented Programming (OOP)  
- File I/O  
- Error handling  
- List operations  
- Menu-driven CLI design
